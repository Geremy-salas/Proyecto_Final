Instrucciones para la Detección de Phishing

Bienvenido a nuestro sistema avanzado de detección de phishing. Aquí te enseñamos cómo usarlo para proteger tu identidad digital y tus datos personales.

  1. Carga imágenes sospechosas para su análisis: Si sospechas que una imagen puede estar relacionada con un intento de phishing (por ejemplo, imágenes de correos electrónicos falsos, sitios web fraudulentos, o links sospechosos), puedes cargar varias imágenes para que nuestro sistema las examine. Haz clic en el siguiente enlace para cargar tus imágenes y permitir que el sistema las analice en busca de patrones de phishing: Cargar Imágenes.
  2. Realiza búsquedas específicas: Si tienes fotos guardadas y deseas realizar una búsqueda rápida de posibles amenazas, puedes buscar por palabras clave. Por ejemplo, si deseas encontrar imágenes relacionadas con correos electrónicos fraudulentos o phishing por su contenido, puedes buscar palabras específicas dentro de tus fotos almacenadas. Haz clic aquí para comenzar a buscar: Buscar Imágenes.
